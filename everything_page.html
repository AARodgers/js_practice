An object has properties and properties that are functions are called methods. Here is a guitar object.

var guitar = { bodyColor: "black", scratchPlateColor: "white", numberOfStrings: 6, brand: "Yamaha", bodyType: "solid", strum: function() {...}, tune: function() {...} };

Defining an object with object literals is most commonly used.

var person = { eyes: 2, feet: 2, hands: 2, eyeColor: "blue" };

Defining objects with an Object constructor

var person = new Object(); person.feet = 2; person.name = "Sandy"; person.hair = "black";

Retrieving and Setting Object Properties

To create a new property called firstName in the person object or to modify the value of an existing firstName property, use dot notation.

person.firstName = "Glenn";

To retrieve the value of a property using dot notation

var fullname = person.firstName + person.lastName;

To write the value of a person.firstName to HTML doc

document.write(person.firstName);

Using square bracket notation to get and set properties

person["firstName"] = "Iggy";

With square bracket you can use variables inside square-bracket when you don't know the name of the property you want to retrieve.

Below is same as above, but has a variable inside the sq bracket rather than a literal string

var personProperty = "firstName";

person[personProperty] = "Iggy";

You can delete properties using the delete operator.

var myObject = { var1: "the value", var2: "another value", var3: "yet another" }; delete myObject.var2;
